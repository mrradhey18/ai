How to Use

Step-by-step guide for agency operators.

🚀 Adding a New Client
1
Duplicate the folder data/clients/dr-sharma-homeo/ → rename to the new client's slug (e.g. city-homeo-clinic).
2
Open profile.json → change name, slug, googleReviewUrl, city, area, services, keywords.
3
Add the new slug + name to the knownClients array inside admin/js/admin.js.
4
Use QR Generator tab to create the patient URL and print the QR code.
✏️ Editing Phrases
1
Open data/phrases/[language]/[type].json in any text editor.
2
Add new phrases to the correct star pool (5star, 4star, 3star). Keep the JSON format valid.
3
Use {{service}}, {{city}}, {{area}} as placeholders — the engine replaces these automatically.
4
Use Phrase Browser tab to verify changes after saving.
🌐 Deployment Checklist
✓
Deploy via Vercel (drag and drop the folder) or GitHub Pages (push to repo).
✓
Patient URL format: https://yourdomain.com/public/index.html?clinic=SLUG
✓
Admin URL: https://yourdomain.com/admin/index.html — not linked anywhere public.
✓
Replace all REPLACE_WITH_REAL_PLACE_ID values in profile JSONs with actual Google Place IDs.
